# AI-Smart-Traffic-Controller

## About
AI-Based Adaptive Smart Traffic Junction Controller using Python simulation.

## What it does
- Simulates a 4-way traffic junction (North, South, East, West)
- AI dynamically assigns green signal time based on vehicle count
- Compares AI controller vs Fixed timer system
- Shows results and graph

## Tools Used
- Python 3.x
- Tkinter (GUI)
- Matplotlib (Graph)

## How to Run
1. Install matplotlib: pip install matplotlib
2. Open traffic.py in Python IDLE
3. Press F5 to run
4. Click "Run Simulation" button
5. Click "Show Graph" button

## Result
AI controller reduces vehicle wait time compared to fixed timer system.
